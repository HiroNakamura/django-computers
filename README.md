# django-computers
Administrador de computadoras y usuarios con **Django**. Es la versiÃ³n en Django de admincomputers de **Spring Boot**.


Es necesario tener **Django Framework** instalado:

```
>>> import django
>>> print(django.get_version())
```

En caso de no tenerlo instalado:
```
#Linux
$ python -m pip install Django

#Windows
$ python -m pip install Django
```

Otra forma:
```
$ pip  install virtualenv
$ mkdir dev
$ cd dev
$ ls
$ pwd
$ mkdir newenv
$ cd newenv
$ virtualenv .
$ .\Scripts\activate
$ pip install django
$ pip freeze
$ .\Scripts\deactivate
$ .\Scripts\activate
```
